How to creat local repository

  1. Right click IT folder using the mouse
  2. Select "Git Bash Here"
  3. Execute "mkdir folderName "
  4. Exeucte "cd folderName "
  5. Execute "git init"

      6. How to link local repo whit GitHub

  6. Go to New pepository page.
  7. Fill pepositoryname field and press "Greate pepository" button.
  8. Execute "git remote add origin https://github.com/ uerName/ folderName. git"
  9. Execute "git branch -M main"
  10. Execute "git push -u origin main"
